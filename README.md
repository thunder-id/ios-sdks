iOS SDK for ThunderID. Provides authentication and user management for native iOS and macOS applications.

## Installation

### Swift Package Manager

```swift
.package(url: "https://github.com/thunder-id/thunderid-ios", from: "0.1.0")
```

Or add the package URL directly in Xcode via **File > Add Package Dependencies**.

### CocoaPods

```ruby
pod 'ThunderID'
```
